baremetal-lifecycle has been installed.

The chart creates:
- A Node CR (synced to Ironic via rest-dynamic-controller)
- Jobs for state transitions (manage, inspect, provide, deploy, undeploy)

State flow: enroll -> manage -> manageable -> (inspect?) -> provide -> available -> deploy -> active

Inspect then provide: Deploy with runInspect=true, wait for ironic-inspect Job, then upgrade with runInspect=false.

To deploy an image, set instance_info in values before the node reaches 'available'.
