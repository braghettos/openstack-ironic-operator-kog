baremetal-lifecycle has been installed.

The composition creates:
- A Node CR (synced to Ironic by KOG / rest-dynamic-controller)
- A single, idempotent provisioner Job that drives the standalone Ironic state machine:
    enroll --manage--> manageable [--inspect--> manageable] --provide--> available --deploy--> active
  The Job runs the whole flow in one pass and STOPS at 'active' (it never undeploys).

Watch progress:
  kubectl -n {{ .Release.Namespace }} logs -f job/ironic-provision-{{ .Release.Name }}
  kubectl get nodes.baremetal.ogen.krateo.io {{ .Values.nodeName }} -o jsonpath='{.status.provision_state}'

Set instance_info.image_source (and image_checksum) in values to deploy an image.
Set runInspect=true to run inspection before provide.
